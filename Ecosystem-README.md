# 🏛️ Agri-Intelligence Ecosystem for ASAL Regions

This is a unified, enterprise-grade agricultural monitoring and decision-support suite designed for Arid and Semi-Arid Lands (ASAL). The ecosystem bridges the gap between tactical farm-level operations and strategic regional governance.

## 🚜 System Components

| Platform | Role | Target User | URL |
| :--- | :--- | :--- | :--- |
| **AGRI-GATEWAY** | Cinematic Portal | Master Entry | `http://localhost:5000` |
| **ShambaIQ** | Tactical Monitoring | Farmers / Field Agents | `http://localhost:5173` |
| **ASAL Platform** | Strategic Command | NGOs / County Govt | `http://localhost:5174` |

---

## 🛠️ How to run the Ecosystem

To run the full "Triple-Node" experience, open three terminals:

### Terminal 1: The Gateway (Master Entrance)
```bash
cd AGRI-GATEWAY
npm run dev
```

### Terminal 2: Tactical Front (ShambaIQ)
```bash
cd SHAMBAIQ
npm run dev
```

### Terminal 3: Strategic Hub (ASAL)
```bash
cd "ASAL-agri intelligence"
npm run dev
```

## 🚀 Key Features

### 🏛️ AGRI-GATEWAY (The Front Door)
*   **Cinematic Obsidian Portal**: A high-fidelity, glassmorphic entry point for the whole suite.
*   **Unified Ticker**: Live simulation of satellite handshakes and regional health status.

### 📡 ShambaIQ (The Sensors)
*   **Drone Survey 3D**: Real-time click-to-target field scouting with telemetry popups.
*   **AI Treatment Plans**: Genetic-specific crop advisory.

### 🏛️ ASAL Intelligence (The Brain)
*   **Biosecurity Foresight**: 3D spatiotemporal simulation of pest and disease spread vectors.
*   **GIS Heatmaps**: Macro-view of vegetation and water scarcity.
